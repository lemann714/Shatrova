0) заполнить csv файлами папку data, закинуть репу на виртуалку
1) файл train.py - его запускаем на виртуалке и ожидаем обученные модели (python3 train.py&)
2) как отбежит обучение, вас заинтересует файл test.py - демонстрирует инференс на ансамбле из двух обученных моделях

Как только на виртуалке закончится обучение помещаем берем от туда файлы
- scaler_iso.pkl
- iso_forest.pkl
- scaler_lstm.pkl
- lstm_ae.pt

и помещаем в репозиторий 

обучение мониторим через ps aux | grep python или top. По оценкам не сориентирую, но по лог файлу monitor.log и сетке параметров можно примерно прикинуть.
